Foreword

The target of this project is to create simple in use yet powerful composing program.
I decided to start this project when I read much about trackers' problems in producing professional sound. I started gathering information about sound processing then...
By now I feel that I have rather enough info to make not just a copy of other tracker programs but to bring something new for both trackers/demomakers and professional composers (who will still work with professional equipment anyway )
